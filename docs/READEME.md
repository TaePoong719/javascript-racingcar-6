# 기능 목록
### oneGameStart
- 하나의 게임을 실행한다
- 각 자동차 별로 moveForwardByRandom를 이용하여 전진 시킨다 
- printRacing을 통해 결과를 출력한다

### moveForwardByRandom
- 0 ~ 9사이 무작위 값을 통하여 무작위 값이 4이상인지 판별한다
- 출력 : moveForward(boolean)

### printRacing
- 각 자동차 별 배열을 통해 출력 포맷에 맞춰 출력한다
- 출력 : result(string)
